Feature Status

Complete status table for every feature in simple-3d. Green = done and stable. Blue = work in progress. Gray = planned but not yet started.

Legend

✅ Done

Implemented, tested, and in the stable public API. Safe to use in production game code.

🔄 In Progress

Partially implemented or under active testing. API exists but may have edge-case issues.

📋 Planned

Designed and scheduled but not yet started.

All Features

FeatureStatusKey API / Notes
Core Engine
Core game loop ✅ Subclass Game; override Start(), Update(float dt), Stop(). dt clamped to 0.1 s.
Entity system (PIMPL) ✅ CreateEntity("name"), FindEntity(name), DestroyEntity(e). Internal Urho3D types never leak into the public API.
Entity hierarchy ✅ CreateChild("wheel"), GetParent(), GetChildren(). Local transforms inherit from parent. Destroy cascades to the whole subtree.
Entity tagging ✅ AddTag("enemy"), HasTag("enemy"), FindEntitiesByTag("coin"). Multiple tags per entity.
SetActive / IsActive ✅ entity->SetActive(false) hides the entity and suspends its scripts and physics. IsActive() reads the current state.
SetTimeScale ✅ SetTimeScale(0.5f) slows all physics and animation. 0 = pause. 1 = normal. Affects dt delivered to Update().
Cursor control ✅ SetCursorLocked(true), SetCursorVisible(false). Used for FPS mouse-look and full-screen gameplay.
Screenshot ✅ TakeScreenshot("shot.png") saves the current frame to disk as PNG.
ResourceDiagnostics ✅ Runtime report of loaded textures, models, and sounds via PrintResourceDiagnostics() or the debug overlay.
3D Rendering
3D models ✅ AddModel("Models/box.mdl", "Materials/Stone.xml"). Accepts .mdl, .obj, .fbx. Convert with Urho3D AssetImporter.
Directional light ✅ AddDirectionalLight(color, intensity). Scene-wide sunlight. Supports shadow mapping.
Point light ✅ AddPointLight(color, range). Range in world units. Falls off to zero at the boundary.
Materials ✅ SetMaterialColor(color), SetTileTexture(path, scaleU, scaleV). Applied per-entity without modifying the material asset on disk.
SkyDome ✅ CreateSkybox("Textures/Skybox/day.xml"). Cubemap-based. One active sky at a time.
Fog ✅ CreateFog(color, start, end). Linear depth fog between world-unit distances.
Ambient light ✅ SetAmbientColor(Color). Scene-wide base illumination added to all surfaces.
Debug draw ✅ DebugDrawLine(start, end, color), DebugDrawBox(pos, size). Active only when SIMPLE3D_DEBUG=1.
Billboard ✅ AddBillboard(texturePath). Texture quad that always faces the camera. Used for icons and health bars in 3D space.
Mc3Bridge scene import/export ✅ Import and export scenes via the Mc3Bridge format. Enables visual scene composition outside the engine with round-trip editing.
Camera
Third-person follow camera ✅ camera.SetFollowTarget(entity, height, distance). Lerp-smooth via SetFollowSpeed(). Camera stays behind last movement direction.
FPS camera ✅ camera.SetFPSTarget(entity, headOffset). Captures mouse for look. SetFPSSensitivity(x, y). camera.Detach() releases mouse.
Orbit camera ✅ camera.SetOrbitTarget(entity). RMB-drag orbits; scroll wheel changes distance. Pitch limits: SetOrbitPitchLimits(min, max).
Orthographic camera ✅ camera.Setup2D(pixelsPerUnit). Used for 2D games and minimaps. SetOrthoSize(height) for manual control.
Split-screen ✅ camera.SetViewport(index, x, y, w, h) — normalized 0.0–1.0 coords. Up to 2 viewports, each with its own Camera object.
3D Physics (Bullet)
RigidBody ✅ AddRigidBody(mass). mass=0 = static (immovable). Simulation at 60 Hz fixed step, independent of render framerate.
Box collider ✅ AddBoxCollider(Vector3 size). Full extents (not half-extents). Centered on entity origin.
Capsule collider ✅ AddCapsuleCollider(radius, height). Total height including hemisphere caps. Recommended for characters.
Sphere collider ✅ AddSphereCollider(radius). Lowest-cost shape. Ideal for rolling objects and projectiles.
Mesh collider ✅ AddMeshCollider("Models/level.mdl"). Triangle mesh, accurate to the model. Static bodies only.
Trigger volumes ✅ AddTriggerBox/Sphere/Capsule(size). SetOnTriggerEnter/Exit(cb). Callbacks receive Entity* other resolved via the entity registry.
Collision layers ✅ SetCollisionLayer(int), SetCollisionMask(int). Bit-flag system. Convention: 1=actors, 2=static geometry, 4=trigger volumes.
Physics constraints ✅ CreateConstraint(type, entityA, entityB). Types: BallSocket, Hinge, Slider, Fixed. Hinge and Slider support motor (SetMotorEnabled, SetMotorSpeed) and limits.
CharacterController ✅ AddCharacterController(radius, height). Capsule body + downward raycast for ground detection. IsOnGround(), Jump(speed), Move(vel, dt).
Ray picking ✅ PickEntity(screenX, screenY), PickPosition(screenX, screenY). Casts a ray from the camera through the given screen point.
2D Physics (Box2D)
2D RigidBody ✅ AddRigidBody2D(BodyType2D::Dynamic). Body types: Dynamic, Static, Kinematic. Operates on the XY plane.
2D box collider ✅ AddCollisionBox2D(width, height). Optional center offset. Supports rotation via entity transform.
2D circle collider ✅ AddCollisionCircle2D(radius). Lowest-cost 2D shape. Recommended for balls, characters, coins.
2D Rendering
StaticSprite2D ✅ AddSprite2D("Sprites/coin.png"). PNG/JPG. Rendered at entity world position. Scale via SetScale(Vector3).
AnimatedSprite2D (Spriter) ✅ AddAnimatedSprite2D("Sprites/player.scml"). Spriter/SCML format. PlayAnimation2D(name, loop), SetFlipX2D(bool), SetFlipY2D(bool).
Tiled tile maps (.tmx) ✅ LoadTileMap("Maps/world1.tmx"). Tiled editor TMX format. Orthogonal tile layers. Returns an Entity* holding the TileMap2D component.
Animation
AnimationController ✅ AddAnimationController(). Play(name, layer, looped, fadeTime), Stop(name), StopAll(fadeTime), SetWeight(name, w), SetSpeed(name, mult), IsPlaying(name), GetTime(name).
AnimStateMachine ✅ Declarative state machine. AddState(name, animPath, loop), AddTransition(from, to, cond, blendTime). Conditions: SetBool, SetFloat, SetTrigger. Auto-transitions; drives AnimationController internally.
Tween ✅ Tween(entity, property, target, duration, easing). 4 easing modes: Linear, EaseIn, EaseOut, EaseInOut. Framerate-independent.
AI & Navigation
Recast/Detour navigation ✅ BuildNavMesh(agentRadius, agentHeight, cellSize). Bakes walkable surface from static geometry. FindPath(entity) and FindPath(Vector3) return waypoints. MoveToward(target, speed) follows the path and returns true on arrival.
BehaviorFSM ✅ AddBehaviorFSM(). Custom states with onEnter/onUpdate/onExit callbacks plus transition conditions. Built-in nav-backed states: AddNavPatrolState(waypoints, speed), AddNavChaseState(target, speed, stopDist).
Audio
Music streaming ✅ PlayMusic("Music/theme.ogg", loop). OGG Vorbis / WAV. Streams from disk; not loaded into RAM. One track at a time.
One-shot SFX ✅ PlaySound("Sounds/jump.wav", volume). Cached after first load. Up to 32 simultaneous channels (Urho3D default).
Per-channel SFX ✅ PlaySoundOnChannel(channel, path, volume). Direct control over specific mixer channels for looping ambient sounds or layered effects.
AudioSource (spatial 3D) ✅ AddAudioSource() on an entity. Play(path), SetGain(0..1), SetRange(near, far), SetPitch(mult). SetAudioListener(entity) positions the listener (typically the camera).
Input
Keyboard ✅ IsKeyDown(Key::W) (held every frame), IsKeyPressed(Key::Space) (single frame on transition). Full Key enum.
Mouse ✅ GetMousePosition() (screen pixels), GetMouseDelta() (pixels since last frame), IsMouseButtonDown/Pressed(btn). 0=left, 1=right, 2=middle.
Touch (multi-touch) ✅ GetTouchCount(), GetTouchPosition(i), GetTouchDelta(i), GetTouchPressure(i). Index 0 = first finger.
Gamepad input ✅ IsGamepadConnected(index), GetGamepadAxis(axis, index), IsGamepadButtonDown/Pressed(btn, index). Configurable deadzone. SetGamepadRumble(low, high, durationSec). Hot-plug connect/disconnect callbacks.
Gamepad focus navigation ✅ UI elements receive gamepad D-pad and stick navigation focus. SetFocusable(true), SetNextFocus(dir, element). Enables controller-only UI traversal.
InputActions ✅ BindAction("Jump", Key::Space, GamepadButton::A), BindAxis2D("Move", Key::W/A/S/D, GamepadAxis::LeftX/Y). Query: IsActionDown/Pressed(name), GetAxis2D(name). Bindings persist to disk.
UI System
Label (HUD text) ✅ CreateLabel(). SetText(str), SetPosition(x, y), SetFontSize(px), SetColor(Color), SetVisible(bool). Position (0,0) = top-left.
Image (icon/background) ✅ CreateImage(texturePath). Screen-space texture quad. SetRect, SetColor, SetVisible(bool). For life icons, health bar backgrounds, splash screens.
Panel (container) ✅ CreatePanel(). Groups child elements; show/hide the entire group at once. Transparent or opaque background color.
Button ✅ CreateButton(label). SetText(str), SetSize(w, h), SetPosition(x, y), SetOnClick(callback). Hover and press states handled automatically.
ProgressBar ✅ CreateProgressBar(). SetValue(0..1), SetColor(Color). Horizontal or vertical orientation. For health, mana, and loading screens.
Slider ✅ CreateSlider(). SetRange(min, max), GetValue(), SetOnChange(callback). Draggable float value for options menus.
UI parent-child hierarchy ✅ panel->AddChild(button). Position and visibility cascade to children. Destroy parent removes all children.
Anchor system ✅ SetAnchor(Anchor::TopLeft / Center / BottomRight / …) + pixel offset. UI elements adapt to different window sizes without manual recalculation.
ZOrder ✅ SetZOrder(int). Higher values render on top. Required for modal dialogs layered over the HUD.
Lua Scripting
Lua scripting (full API) ✅ entity->AddScript("Scripts/player.lua"). LuaJIT / Lua 5.1. Lifecycle: Start(), Update(dt), Stop(), FixedUpdate(dt). Full simple-3d API bound. Liveness guards prevent use-after-free. All bindings are type-checked at call time. See Lua API for the full reference.
Networking (ENet)
ENet networking ✅ UDP client/server via ENet. StartServer(port, maxClients), Connect(host, port). Automatic scene entity replication (SetReplicated(true)). Custom game messages (IDs 100–32767): Broadcast, SendToClient, SendToServer. Connection callbacks: OnClientConnected, OnClientDisconnected, OnServerConnected, OnServerDisconnected, OnMessage.
Effects
ParticleEmitter ✅ AddParticleEmitter(xmlPath). Play(), Stop(), Burst(count), IsEmitting(). 5 built-in preset XMLs: Sparkle, Smoke, Explosion, Dust, Stars.
Scene & Game Management
Scene management ✅ LoadScene(path), FadeOut(duration, cb), FadeIn(duration), SetLevelList(paths), LoadNextLevel(). Camera and audio survive scene transitions; all other entities are destroyed and re-created.
SaveData (JSON multi-slot) ✅ Set(key, val), Get<T>(key, default), Save(slot), Load(slot), Delete(slot), Exists(slot). JSON format. Platform-correct paths: Linux ~/.local/share/<app>/, Windows %APPDATA%\<app>\.
