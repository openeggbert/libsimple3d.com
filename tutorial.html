Tutorial Series

30-part tutorial series from Hello World to Networking.

Step-by-step guides covering every major feature of simple-3d. Each part builds on the last — start at Part 01 or jump to any topic you need.

Beginner Parts 01–06 Intermediate Parts 07–29 Advanced selected parts C++23 Lua

Getting Started

Parts 01–06
No prior simple-3d experience required. These six parts take you from an empty project to a scene with physics, a follow camera, and your first Lua script.
Part 01 Beginner

Hello World

Your first game: subclass Game, override Start() and Update(), and run.
Part 02 Beginner

Lights and Colors

Illuminate the scene with AddDirectionalLight(Color, intensity) and tweak color temperatures.
Part 03 Beginner

Keyboard Input

IsKeyDown() for held state every frame vs IsKeyPressed() for the first frame only.
Part 04 Beginner

3D Physics

Add Bullet rigid bodies with AddRigidBody(mass) — pass 0.0f for static world geometry.
Part 05 Beginner

Third-Person Camera

Smooth third-person view with camera->Follow(entity, distance, height) and collision avoidance.
Part 06 Beginner

Lua Scripting Basics

Attach per-entity Lua logic with entity->AddScript(path) and implement Start() / Update(dt).

Core Systems

Parts 07–20
The full feature set. Animation, audio, 2D mode, navigation, gamepad, input remapping, UI, save data, and scene transitions. Some parts cover advanced API patterns and are marked accordingly.
Part 07 Advanced

Character Controller

Capsule-based movement and ground detection via entity->AddCharacterController(radius, height).
Part 08 Intermediate

Skeletal Animation

Blend and layer skeletal clips on a character mesh using entity->AddAnimationController().
Part 09 Advanced

Animation State Machine

Data-driven state transitions and blending with entity->AddAnimStateMachine().
Part 10 Intermediate

Audio

Stream background music and trigger one-shot effects via PlayMusic(path) and PlaySound(path).
Part 11 Intermediate

Spatial Audio

3D positional sound with distance attenuation using entity->AddAudioSource().
Part 12 Intermediate

2D Mode

Switch to a flat, pixel-perfect view using camera->SetOrthographic(true) and work with sprites.
Part 13 Intermediate

2D Physics

Box2D rigid bodies alongside the 3D physics world via entity->AddRigidBody2D(mass).
Part 14 Advanced

Navigation and Pathfinding

Bake a Recast nav-mesh with BuildNavMesh() in Start() and steer agents automatically.
Part 15 Intermediate

Gamepad Input

Detect connected controllers with IsGamepadConnected(index) and read axes and buttons.
Part 16 Advanced

Input Actions

Rebindable cross-device inputs via BindAction("Jump").Key(Key::Space).GamepadButton(GamepadButton::A).
Part 17 Intermediate

UI System

Build HUDs and menus with CreateLabel(), CreateImage(), CreatePanel(), and CreateButton().
Part 18 Intermediate

UI Advanced

Progress bars and styled overlays with CreateProgressBar(), SetValue(0–1), and SetColor().
Part 19 Intermediate

Save and Load

Persist and restore game state across sessions through GetSaveData() / SaveData.
Part 20 Intermediate

Scene Management

Load and transition between levels at runtime using LoadScene(path).

Advanced Topics

Parts 21–30
Polish, performance, and multiplayer. Particle effects, entity hierarchies, joints, triggers, AI, ray picking, debug utilities, tweens, atmosphere, and networking.
Part 21 Intermediate

Particle Systems

Spawn and configure visual effects via entity->AddParticleEmitter(xmlPath).
Part 22 Advanced

Entity Hierarchy

Parent-child transform chains and scene graphs with entity->CreateChild(name).
Part 23 Advanced

Physics Constraints

Rigidly join two bodies with joints such as entity->AddBallSocketConstraint(other).
Part 24 Intermediate

Collision Triggers

Overlap enter/exit callbacks without physics response via entity->SetTrigger(true).
Part 25 Advanced

AI Behavior FSM

Rule-based NPC logic and transitions with entity->AddBehaviorFSM() and custom states.
Part 26 Intermediate

Ray Picking

Click-to-select scene objects via PickEntity(GetMousePosition()) and ray casting.
Part 27 Intermediate

Debug Draw

Visualise colliders, nav-mesh, and paths at runtime with SetDebugDrawEnabled(true).
Part 28 Intermediate

Tweens

Animate properties smoothly over time with CreateTween(duration) and chained actions.
Part 29 Intermediate

Sky and Atmosphere

Skydome backgrounds and atmospheric fog via SetSkyDome(texPath) / SetSkyDomeEnabled().
Part 30 Advanced

Networking Basics

ENet client/server setup and automatic entity replication via GetNetworkManager().
